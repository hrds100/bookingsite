# NFStay Booking Site

Standalone vacation rental booking platform at **nfstay.app**.

## Quick Start

```bash
npm install
npm run dev      # → http://localhost:5173
```

## Stack

React 18 + Vite + TypeScript + Tailwind CSS + shadcn/ui
